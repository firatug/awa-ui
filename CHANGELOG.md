# Changelog

All notable changes to **AWA UI** are documented in this file.

## [1.0.0] — 2026-07-15

### Added

- Initial release of AWA UI — Universal Admin Design System
- Design tokens (CSS, TypeScript, JSON)
- Theme provider with light / dark / system modes
- Live theme customizer (sidebar, density, radius, font, RTL, primary color)
- App shell: sidebar (default / compact / icon / hover), header, command palette
- Core UI primitives (Button, Input, Select, Modal, Drawer, Tabs, Table, …)
- Data table system with sorting, filtering, selection, pagination
- Chart components (Area, Line, Bar, Donut, Sparkline)
- Six dashboard demos (Overview, Analytics, SaaS, CRM, Projects, Finance)
- Auth templates, settings, users & roles, apps, error pages
- Design System page, Typography page, Component Playground
- English and Turkish locales
- Unit tests for critical components
- Documentation and Implementation Guide
