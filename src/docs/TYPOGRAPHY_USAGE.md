# Typography usage rules

1. Default font is configured via preferences / `--font-sans` (Manrope by default).
2. Display/H1–H6 map to `--text-display` … `--text-h6`.
3. Body copy uses `--text-body` / `--text-body-sm`; captions use `--text-caption`.
4. Labels use `--text-label` + medium weight.
5. Code and mono snippets use `--font-mono`.
6. Keep line length comfortable; long paragraphs use `--leading-relaxed`.
